Twitch Monitor

Twitch is a live streaming videa site specializing in video game related content. The Twitch Monitor project displays a list of twitch channels and their current status. Clicking on a channel entry will take the user to the Twitch page for that channel. User may also search for a channel, which adds an entry to the list.

Implemented this project using Twitch Web API to search for and get status of Twitch channels. Used JQuery funtions to perform AJAX calls to Twitch API, then converted JSON response to render channel entry on page.

Wikipedia Viewer

Wikipedia Viewer project for FreeCodeCamp. User can enter search string and pick from a list of resulting topics, or press the dice icon to get random list of topics. Click a topic in the list to see Wikipedia page for that topic.

Uses Wikipedia Web API for search. API calls made using JQuery functions. Also uses JQuery animation functions to move elements on page.
